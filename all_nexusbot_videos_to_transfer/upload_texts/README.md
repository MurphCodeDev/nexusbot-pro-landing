# Upload Texts

Each `.md` file matches one video file in the transfer package.

Before publishing, replace:

```text
[WEBSITE LINK]
```

with your real Cloudflare Pages landing URL.

Suggested use:
- Use `Title` as the video title.
- Use `Description` as the YouTube/TikTok/X description or caption.
- Use `Pinned Comment` as the first pinned comment where the platform supports it.
